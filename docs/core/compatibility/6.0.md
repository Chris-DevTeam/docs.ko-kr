# <a name="breaking-changes-in-net-60"></a>.NET 6.0의 호환성이 손상되는 변경

앱을 .NET 6.0으로 마이그레이션하는 경우 여기에 나열된 호환성이 손상되는 변경이 영향을 줄 수 있습니다. 변경 내용은 ASP.NET Core 또는 Windows Forms와 같은 기술 영역별로 그룹화됩니다.
