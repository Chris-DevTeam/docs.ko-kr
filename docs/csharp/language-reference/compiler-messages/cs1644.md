# <a name="compiler-error-cs1644"></a>컴파일러 오류 CS1644

'feature' 기능은 표준화된 ISO C# 언어 사양의 일부가 아니므로 다른 컴파일러에서 지원하지 않을 수도 있습니다.  
  
 이 오류는 [**LangVersion**](../compiler-options/language.md#langversion) 옵션 ISO-1을 지정하고 컴파일 중인 코드에서 ISO 1.0 표준에 포함되지 않는 기능을 사용할 때 발생합니다. 이 오류를 해결하려면 제네릭 또는 무명 메서드와 같은 C# 2.0 컴파일러 기능 중 어떤 것도 ISO-1 호환성 옵션과 함께 사용하지 않아야 합니다.  
  
 다음 샘플에서는 CS1644 오류가 발생합니다.  
  
```csharp  
// CS1644.cs  
// compile with: /langversion:ISO-1 /target:library  
class C<T> {}   // CS1644  
```
