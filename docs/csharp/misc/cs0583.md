# <a name="compiler-error-cs0583"></a>컴파일러 오류 CS0583

내부 컴파일러 오류입니다. 컴파일러에서 내부 오류가 발생했습니다. 이 문제를 해결하려면 아래 목록에 나오는 위치 부근의 프로그램을 단순화하거나 변경합니다. 목록 맨 위에 있는 위치가 내부 오류의 발생 지점에 더 가깝습니다.

> [!NOTE]
> 이 컴파일러 오류는 Roslyn에서 더 이상 사용되지 않습니다.
