# <a name="compiler-error-cs2012"></a>컴파일러 오류 CS2012

'file'을 쓰기용으로 열 수 없습니다.  
  
`-bugreport:`, 컴파일러 옵션을 사용 하는 동안 `file` 파일을 쓰기용으로 열 수 없습니다. 올바른 파일 이름을 지정했는지, 파일이 읽기 전용이 아닌지 확인합니다.

> [!IMPORTANT]
> `bugreport`옵션은 더 이상 지원 되지 않습니다.
