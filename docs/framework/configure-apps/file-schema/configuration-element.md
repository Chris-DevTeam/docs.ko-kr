# <a name="configuration-element"></a>\<configuration> 요소

공용 언어 런타임 및 .NET Framework 애플리케이션에서 사용하는 모든 구성 파일의 루트 요소입니다.

**\<configuration>**

## <a name="syntax"></a>구문

```xml
<configuration>
  <!-- Configuration settings -->
</configuration>
```

## <a name="attributes"></a>특성

None

## <a name="parent-element"></a>부모 요소

None
