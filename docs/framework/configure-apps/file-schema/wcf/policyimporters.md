# \<policyImporters>

이 구성 요소는 바인딩에 대한 사용자 지정 정책 어설션의 가져오기를 제어하는 모든 정책 가져오기 지정합니다. 각 자식 요소는 `policyImporter` 바인딩 기능에 대 한 사용자 지정 정책 어설션을 검색 하 고 어설션에 필요한 기능을 구현 하는 사용자 지정 바인딩 요소를 연결 하는 데 사용 되는 <>입니다.
