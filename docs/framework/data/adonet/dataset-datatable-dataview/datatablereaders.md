 **Datatable** 에서 **DataTableReader** 을 만드는 경우 결과 **DataTableReader** 개체는 삭제 된 것으로 표시 된 행을 제외 하 고 생성 된 **datatable** 과 동일한 데이터를 포함 하는 하나의 결과 집합을 포함 합니다. 열은 원래 **DataTable** 과 동일한 순서로 표시 됩니다.
