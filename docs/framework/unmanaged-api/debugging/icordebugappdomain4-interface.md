# <a name="icordebugappdomain4-interface"></a>ICorDebugAppDomain4 인터페이스

ICorDebugAppDomain 인터페이스를 논리적으로 확장하여 COM 호출 가능 래퍼에서 관리되는 개체를 가져옵니다.  
  
## <a name="method"></a>메서드  
  
|메서드|설명|  
|------------|-----------------|  
|[GetObjectForCCW 메서드](icordebugappdomain4-getobjectforccw-method.md)|CCW(COM 호출 가능 래퍼) 포인터에서 관리되는 개체를 가져옵니다.|
