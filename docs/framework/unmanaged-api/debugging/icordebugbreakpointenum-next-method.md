# <a name="icordebugbreakpointenumnext-method"></a>ICorDebugBreakpointEnum::Next 메서드

현재 위치에서 시작 하 여 열거형에서 지정 된 수의 ICorDebugBreakpoint 인스턴스를 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT Next (  
    [in] ULONG  celt,  
    [out, size_is(celt), length_is(*pceltFetched)]  
        ICorDebugBreakpoint *breakpoints[],  
    [out] ULONG *pceltFetched  
);  
```  
  
## <a name="parameters"></a>매개 변수  

 `celt`  
 진행 `ICorDebugBreakpoint` 검색할 인스턴스 수입니다.  
  
 `breakpoints`  
 제한이 각각 중단점을 나타내는 개체를 가리키는 포인터의 배열입니다 `ICorDebugBreakpoint` .  
  
 `pceltFetched`  
 제한이 실제로 반환 된 인스턴스 수에 대 한 포인터 `ICorDebugBreakpoint` 입니다. 이 일 경우이 값은 null 일 수 있습니다 `celt` .
