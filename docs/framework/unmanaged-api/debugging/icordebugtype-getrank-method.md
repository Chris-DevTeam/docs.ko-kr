# <a name="icordebugtypegetrank-method"></a>ICorDebugType::GetRank 메서드

배열 형식의 차원 수를 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT GetRank (  
    [out] ULONG32   *pnRank  
);  
```  
  
## <a name="parameters"></a>매개 변수  

 `pnRank`  
 제한이 차원 수에 대 한 포인터입니다.
