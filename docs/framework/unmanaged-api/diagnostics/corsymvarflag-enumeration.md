# <a name="corsymvarflag-enumeration"></a>CorSymVarFlag 열거형

변수가 컴파일러에서 생성 되었는지 여부를 나타냅니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
typedef enum CorSymVarFlag
{  
    VAR_IS_COMP_GEN = 1  
} CorSymVarFlag;  
```  
  
## <a name="members"></a>멤버  
  
|멤버|설명|  
|------------|-----------------|  
|`VAR_IS_COMP_GEN`|지정 된 변수가 컴파일러에서 생성 되었음을 나타냅니다.|  
  
## <a name="requirements"></a>요구 사항  

 **헤더:** CorSym, CorSym
