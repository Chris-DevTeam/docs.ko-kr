# <a name="isymunmanagednamespace-interface"></a>ISymUnmanagedNamespace 인터페이스

## <a name="methods"></a>메서드  
  
|메서드|설명|  
|------------|-----------------|  
|[GetName 메서드](isymunmanagednamespace-getname-method.md)|이 네임 스페이스의 이름을 가져옵니다.|
