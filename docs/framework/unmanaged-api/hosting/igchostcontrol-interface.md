# <a name="igchostcontrol-interface"></a>IGCHostControl 인터페이스

가비지 수집기에서 호스트에 가상 메모리의 제한을 변경 하도록 요청할 수 있는 메서드를 제공 합니다.  
  
## <a name="methods"></a>메서드  
  
|메서드|설명|  
|------------|-----------------|  
|[RequestVirtualMemLimit 메서드](igchostcontrol-requestvirtualmemlimit-method.md)|가상 메모리의 한도를 변경 하도록 호스트에 요청 합니다.|

 **라이브러리:** MSCorEE.dll의 리소스로 포함 됩니다.
