# <a name="imanagedobject-interface"></a>IManagedObject 인터페이스

관리 되는 개체를 제어 하는 메서드를 제공 합니다.  
  
## <a name="methods"></a>메서드  
  
|메서드|설명|  
|------------|-----------------|  
|[GetObjectIdentity 메서드](imanagedobject-getobjectidentity-method.md)|이 관리 되는 개체의 id를 가져옵니다.|
