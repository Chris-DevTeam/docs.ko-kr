# <a name="coiniticor-enumeration"></a>COINITICOR 열거형

공용 언어 런타임을 초기화할 때 [CoInitializeCor](../hosting/coinitializecor-function.md) 에서 사용 하는 상수를 지정 합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
typedef enum tagCOINITCOR  
{  
    COINITCOR = 0x0  
} COINITICOR;  
```
