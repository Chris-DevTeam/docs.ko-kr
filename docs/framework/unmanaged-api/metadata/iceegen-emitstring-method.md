# <a name="iceegenemitstring-method"></a>ICeeGen::EmitString 메서드

지정 된 문자열을 코드 베이스에 내보냅니다.  
  
 이 메서드는 사용 되지 않으므로 사용 하면 안 됩니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT EmitString (  
    [in]  LPWSTR    lpString,  
    [out] ULONG     *RVA  
);  
```  
  
## <a name="parameters"></a>매개 변수  

 `lpString`  
 진행 내보낼 문자열입니다.  
  
 `RVA`  
 제한이 내보낸 문자열의 상대 가상 주소입니다.
