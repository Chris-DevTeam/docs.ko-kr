# <a name="iceegengenerateceefile-method"></a>ICeeGen::GenerateCeeFile 메서드

현재이 [ICeeGen](iceegen-interface.md) 인터페이스로 로드 된 코드 베이스를 포함 하는 코드 베이스 파일을 생성 합니다.  
  
 이 메서드는 사용 되지 않으므로 사용 하면 안 됩니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT GenerateCeeFile ();  
```  
  
## <a name="parameters"></a>매개 변수  

 `GenerateCeeFile` 매개 변수를 사용 하지 않습니다.
