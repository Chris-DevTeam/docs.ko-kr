# <a name="iceegengetstring-method"></a>ICeeGen::GetString 메서드

지정 된 상대 가상 주소에 저장 된 문자열을 가져옵니다.  
  
 이 메서드는 사용 되지 않으므로 사용 하면 안 됩니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT GetString (  
    [in]  ULONG      RVA,
    [out] LPWSTR     *lpString  
);  
```  
  
## <a name="parameters"></a>매개 변수  

 `RVA`  
 진행 반환할 문자열의 상대 가상 주소입니다.  
  
 `lpString`  
 제한이 반환 된 문자열입니다.
