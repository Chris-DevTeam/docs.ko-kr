# <a name="imetadataemit-interface"></a>IMetaDataEmit 인터페이스

현재 정의 된 범위에서 어셈블리에 대 한 메타 데이터를 만들고, 수정 하 고, 저장 하는 메서드를 제공 합니다. 메타 데이터는 메모리에 저장 되거나 디스크에 저장 될 수 있습니다.
