# <a name="imetadataemit2savedelta-method"></a>IMetaDataEmit2::SaveDelta 메서드

현재 편집 하며 계속 하기 세션의 변경 내용을 지정 된 파일에 저장 합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT SaveDelta (  
    [in] LPCWSTR     szFile,
    [in] DWORD       dwSaveFlags  
);  
```  
  
## <a name="parameters"></a>매개 변수  

 `szFile`  
 진행 변경 내용을 저장할 파일 이름입니다.  
  
 `dwSaveFlags`  
 [in] 예약되어 있습니다. 0이어야 합니다.
