# <a name="imetadataimportresetenum-method"></a>IMetaDataImport::ResetEnum 메서드

지정한 열거자를 지정한 위치로 다시 설정합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT ResetEnum (  
   [in] HCORENUM    hEnum,
   [in] ULONG       ulPos  
);  
```  
  
## <a name="parameters"></a>매개 변수  

 `hEnum`  
 진행 다시 설정할 열거자입니다.  
  
 `ulPos`  
 진행 열거자를 배치할 새 위치입니다.
