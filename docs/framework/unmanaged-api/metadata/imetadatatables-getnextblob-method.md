# <a name="imetadatatablesgetnextblob-method"></a>IMetaDataTables::GetNextBlob 메서드

테이블에서 다음 BLOB (binary large object)의 인덱스를 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT GetNextBlob (  
    [in]  ULONG   ixBlob,  
    [out] ULONG   *pNext  
);  
```  
  
## <a name="parameters"></a>매개 변수  

 `ixBlob`  
 진행 Blob의 열에서 반환 된 인덱스입니다.  
  
 `pNext`  
 제한이 다음 BLOB의 인덱스에 대 한 포인터입니다.
