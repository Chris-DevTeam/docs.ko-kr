## <a name="description"></a>Description  

 이 이벤트는 클라이언트 기반의 캐시된 채널 팩터리 수를 보고하도록 내보내집니다.  
  
## <a name="message"></a>메시지  

 캐시된 채널 팩터리 수는 '%1'입니다.  최대 '%2'개 채널 팩터리를 캐시할 수 있습니다.
