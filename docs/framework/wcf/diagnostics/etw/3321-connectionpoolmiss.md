## <a name="description"></a>Description  

 이 이벤트는 풀에 사용할 수 있는 연결이 없는 경우 내보내집니다.  
  
## <a name="message"></a>메시지  

 %1에 대한 풀에 사용 가능한 연결이 없고 %2개의 연결을 사용 중입니다.
