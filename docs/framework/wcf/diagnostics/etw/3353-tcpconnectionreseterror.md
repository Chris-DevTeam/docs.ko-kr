## <a name="description"></a>Description  

 이 이벤트는 TCP 연결 재설정 오류 시에 내보내집니다.  
  
## <a name="message"></a>메시지  

 원격 주소 %2에 대한 SocketId:%1에 연결 재설정 오류가 발생했습니다.
