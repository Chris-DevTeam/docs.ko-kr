|||  
|-|-|  
|ID|3419|  
|키워드|HTTP|  
|Level|오류|

## <a name="description"></a>Description  

 이 이벤트는 WebSocket 연결이 거부될 때 내보내집니다.  
  
## <a name="message"></a>메시지  

 WebSocket 연결이 상태 코드 '%1'(으)로 거부되었습니다.
