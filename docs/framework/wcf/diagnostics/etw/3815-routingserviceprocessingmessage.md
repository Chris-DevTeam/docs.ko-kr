|-|-|

|Level|정보|

## <a name="description"></a>Description  

 이 이벤트는 라우팅 서비스가 메시지를 처리할 때 내보내집니다.  
  
## <a name="message"></a>메시지  

 라우팅 서비스에서 트랜잭션 %4에서 받은 ID: '%1', 동작: '%2', 인바운드 URL: '%3'인 메시지를 처리하는 중입니다.
