|||  
|-|-|

|키워드|RoutingServices|  
|Level|오류|

## <a name="description"></a>Description  

 이 이벤트는 라우팅 서비스에서 중복된 콜백 예외가 발생할 때 내보내집니다.  
  
## <a name="message"></a>메시지  

 라우팅 서비스 구성 요소 %1에 중복 콜백 예외가 발생했습니다.
