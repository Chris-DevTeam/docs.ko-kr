## <a name="properties"></a>속성  
  
|||  
|-|-|  
|ID|3818|  
|키워드|RoutingServices|  
|Level|Error|  
|채널|Microsoft-Windows-애플리케이션 서버-애플리케이션/디버그|  
  
## <a name="description"></a>설명  

 이 이벤트는 라우팅 서비스에서 중복된 콜백 예외가 발생할 때 내보내집니다.  
  
## <a name="message"></a>메시지  

 라우팅 서비스 구성 요소 %1에 중복 콜백 예외가 발생했습니다.
