|||  
|-|-|  
|ID|3823|  
|키워드|RoutingServices|  
|Level|경고|

## <a name="description"></a>Description  

 이 이벤트는 라우팅 서비스가 오류 응답 메시지를 다시 보낼 때 내보내집니다.  
  
## <a name="message"></a>메시지  

 라우팅 서비스에서 동작이 '%1'인 오류 응답 메시지를 다시 보내는 중입니다.
