## <a name="description"></a>Description  

 이 이벤트는 라우팅 서비스에서 기존 트랜잭션을 사용하여 메시지를 보낼 때 내보내집니다.  
  
## <a name="message"></a>메시지  

 라우팅 서비스에서 기존 트랜잭션 '%1'을(를) 사용하여 메시지를 보냅니다.
