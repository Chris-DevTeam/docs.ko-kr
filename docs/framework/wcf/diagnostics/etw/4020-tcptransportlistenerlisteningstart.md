## <a name="description"></a>Description  

 이 이벤트는 TCP 전송 수신기가 수신을 시작할 때 내보내집니다.  
  
## <a name="message"></a>메시지  

 TCP 전송 리스너가 URI '%1'에서 수신을 시작하는 중입니다.
