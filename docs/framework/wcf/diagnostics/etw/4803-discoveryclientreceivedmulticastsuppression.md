## <a name="description"></a>설명  

 이 이벤트는 DiscoveryClient가 DiscoveryProxy로부터 멀티캐스트 비표시 오류(Suppression) 메시지를 받았을 때 내보내집니다.  
  
## <a name="message"></a>메시지  

 DiscoveryClient가 DiscoveryProxy로부터 멀티캐스트 비표시 오류(Suppression) 메시지를 받았습니다.
