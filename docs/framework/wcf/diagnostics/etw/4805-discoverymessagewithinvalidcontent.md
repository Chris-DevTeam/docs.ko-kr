## <a name="description"></a>Description  

 이 이벤트는 검색 메시지에 잘못된 콘텐츠가 포함된 경우에 내보내집니다.  
  
## <a name="message"></a>메시지  

 잘못된 내용이 있어 messageId='%2'인 %1 메시지가 DiscoveryClient에서 삭제되었습니다.
