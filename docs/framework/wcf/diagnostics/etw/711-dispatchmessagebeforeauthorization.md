## <a name="description"></a>Description  

 이 이벤트는 메시지 디스패치에 대한 인증이 시작될 때 내보내집니다.  
  
## <a name="message"></a>메시지  

 메시지 디스패치를 위한 권한 부여를 시작합니다.
