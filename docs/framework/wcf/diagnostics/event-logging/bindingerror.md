# <a name="bindingerror"></a>BindingError

Id: 159  
  
 심각도: 오류  
  
 범주: SMSvcHost  
  
## <a name="description"></a>Description  

 이 이벤트는 특정 사이트의 특정 바인딩을 구문 분석하는 동안 오류가 발생했으므로 사이트에 대한 해당 프로토콜을 임시로 사용할 수 없음을 나타냅니다. 이 이벤트는 바인딩, 소스, 예외, 프로세스 이름 및 프로세스 ID를 나열합니다.
