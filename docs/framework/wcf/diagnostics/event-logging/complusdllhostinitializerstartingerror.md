# <a name="complusdllhostinitializerstartingerror"></a>ComPlusDllHostInitializerStartingError

Id: 131  
  
 심각도: 오류  
  
 범주: ServiceModel

 이 이벤트는 DllHost 이니셜라이저를 시작하는 동안 오류가 발생했음을 나타냅니다. 이 이벤트는 AppID, 예외, 프로세스 이름 및 프로세스 ID를 표시합니다.
