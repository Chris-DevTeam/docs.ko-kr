# <a name="missingnecessarykeyusage"></a>MissingNecessaryKeyUsage

Id: 156  
  
 심각도: 오류  
  
 범주: TransactionBridge  
  
## <a name="description"></a>설명  

 이 이벤트는 지정된 주체 이름 및 지문의 ID 인증서가 자체 KeyUsages의 특정 데이터를 제공하지 않음을 나타냅니다. 여기서는 프로세스 이름과 프로세스 ID도 나열합니다.
