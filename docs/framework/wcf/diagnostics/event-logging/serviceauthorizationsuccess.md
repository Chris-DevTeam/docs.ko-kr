Id: 176  
  
 심각도: 알림

 이 이벤트는 서비스 인증 완료되었음을 나타냅니다. 이벤트에서는 서비스, 동작, 클라이언트 ID, 권한 부여 컨텍스트, 동작 ID 및 ServiceAuthorizationManager를 나열합니다.
