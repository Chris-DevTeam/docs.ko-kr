# <a name="servicestartfailed"></a>ServiceStartFailed

Id: 164  
  
 심각도: 오류  
  
 범주: SMSvcHost  
  
## <a name="description"></a>Description  

 이 이벤트는 서비스를 시작하라는 요청이 실패했음을 나타냅니다. 이 이벤트는 오류 코드, 프로세스 이름 및 프로세스 ID를 표시합니다.
