# <a name="sslnoaccessibleprivatekey"></a>SslNoAccessiblePrivateKey

Id: 155  
  
 심각도: 오류  
  
 범주: TransactionBridge  
  
## <a name="description"></a>Description  

 이 이벤트는 특정 주체 이름 및 지문의 ID 인증서에 액세스할 수 있는 프라이빗 키가 없음을 나타냅니다. 이 이벤트는 프로세스 이름과 프로세스 ID를 표시합니다.
