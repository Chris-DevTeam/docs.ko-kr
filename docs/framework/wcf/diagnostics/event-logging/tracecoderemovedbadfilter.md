# <a name="tracecoderemovedbadfilter"></a>TraceCodeRemovedBadFilter

Id: 114  
  
 심각도: 오류  
  
 범주: ServiceModel  
  
## <a name="description"></a>설명  

 이 이벤트는 메시지 본문에 액세스하고 있던 잘못된 메시지 로깅 필터가 제거되었음을 나타냅니다. 이 이벤트는 XPath, 프로세스 이름 및 프로세스 ID를 나열합니다.
