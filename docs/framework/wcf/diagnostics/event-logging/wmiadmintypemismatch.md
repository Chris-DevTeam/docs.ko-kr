# <a name="wmiadmintypemismatch"></a>WmiAdminTypeMismatch

ID: 128  
  
 심각도: 오류  
  
 범주: ServiceModel  
  
## <a name="description"></a>Description  

 이 이벤트는 유형 불일치가 발생했음을 나타냅니다. 지정한 CIM 클래스 속성에서 특정 유형의 값을 받아들일 수 없습니다. 이 이벤트는 프로세스 이름과 프로세스 ID를 표시합니다.
