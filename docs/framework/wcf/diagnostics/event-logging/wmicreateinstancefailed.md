# <a name="wmicreateinstancefailed"></a>WmiCreateInstanceFailed

ID: 123  
  
 심각도: 오류  
  
 범주: ServiceModel  
  
## <a name="description"></a>Description  

 이 이벤트는 WMI CreateInstance 요청이 처리되지 않았음을 나타냅니다. 이벤트는 클래스 이름, 예외, 프로세스 이름 및 프로세스 ID를 표시합니다.
