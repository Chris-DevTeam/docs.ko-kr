# <a name="wmiexecmethodfailed"></a>WmiExecMethodFailed

Id: 125  
  
 심각도: 오류  
  
 범주: ServiceModel  
  
## <a name="description"></a>설명  

 이 이벤트는 WMI ExecMethod 요청이 처리되지 않았음을 나타냅니다. 이 이벤트는 예외, 프로세스 이름 및 프로세스 ID를 나열합니다.
