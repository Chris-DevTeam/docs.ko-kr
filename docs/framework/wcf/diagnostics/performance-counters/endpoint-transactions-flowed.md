# <a name="endpoint-transactions-flowed"></a>엔드포인트: Transactions Flowed

카운터 이름: Transactions Flowed  
  
## <a name="description"></a>Description  

 이 엔드포인트에서 작업에 적용된 초당 트랜잭션의 수입니다. 엔드포인트에 전송된 메시지에 트랜잭션 ID가 있을 때마다 이 카운터가 증가합니다.
