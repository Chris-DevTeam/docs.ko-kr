# <a name="systemservicemodelchannelspeernodeopenfailed"></a>System.ServiceModel.Channels.PeerNodeOpenFailed

환경 연결을 열 수 없습니다.  
  
## <a name="description"></a>설명  

 이 추적은 환경 연결을 설정하려는 시도가 있은 이후에 발생합니다. 가능한 이유에는 응답하지 않는 확인자, 잘못된 바인딩 설정 또는 일시적인 네트워크 오류 등이 포함됩니다.  
  
## <a name="troubleshooting"></a>문제 해결  

 두 노드 모두 유효한 주소가 지정되어 있고 적절하게 구성되어 있는지 확인합니다.
