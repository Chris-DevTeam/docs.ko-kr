# <a name="behavior-class"></a>Behavior 클래스

## <a name="syntax"></a>구문  
  
```csharp
class Behavior  
{  
  string Type;  
};  
```  
  
## <a name="methods"></a>메서드  

 Behavior 클래스는 메서드를 정의하지 않습니다.  
  
## <a name="properties"></a>속성  

 Behavior 클래스에는 다음과 같은 속성이 있습니다.  
  
### <a name="type"></a>유형  

 데이터 형식: 문자열  
  
 액세스 형식: 읽기 전용  
  
 동작의 형식입니다.
