 선언적으로 코드로 만든 사용자 지정 활동 샘플이 들어 있습니다.
  
 [사용자 지정 활동 디자이너](custom-activity-designers.md)  
 Workflow Designer에 표시할 사용자 지정 디자이너를 사용하는 활동 샘플이 들어 있습니다.
