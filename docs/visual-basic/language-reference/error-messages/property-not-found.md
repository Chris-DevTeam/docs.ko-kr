# <a name="property-not-found"></a>속성을 찾을 수 없습니다.

이 개체는 지정 된 속성을 지원 하지 않습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. 속성 이름의 철자를 확인 합니다.  
  
2. 개체의 설명서를 확인 하 여 개체가 실제로 "caption" 또는 유사한 이름의 속성을 지 원하는 경우 "텍스트" 속성 같은 항목에 액세스 하지 않도록 합니다.
