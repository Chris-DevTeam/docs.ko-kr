# <a name="bc30655-unable-to-find-required-file-filename"></a>BC30655: ' ' 필수 파일을 찾을 수 없습니다. \<filename>

Visual Studio에 필요한 파일이 없거나 손상되었습니다.

 **오류 ID:** BC30655

## <a name="to-correct-this-error"></a>이 오류를 해결하려면

- Visual Studio를 다시 설치합니다.
