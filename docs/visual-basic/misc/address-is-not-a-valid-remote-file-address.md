# <a name="address-is-not-a-valid-remote-file-address"></a>\<address>가 올바른 원격 파일 주소가 아닙니다.

\<address> 은 올바른 원격 파일 주소가 아닙니다. 올바른 주소는 프로토콜, 경로 및 파일 이름을 포함해야 합니다.  
  
 작업 업로드 또는 다운로드에 대해 잘못된 주소가 지정되었습니다. 주소에는 `ftp` 또는 `http`와 같은 프로토콜, 올바른 경로 및 파일 이름이 포함되어야 합니다. 이 오류는 파일 경로를 잘못 구문 분석한 결과 발생할 수 있습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 지정한 주소를 확인하여 잘못된 부분을 확인하고 필요한 정보를 제공합니다.
