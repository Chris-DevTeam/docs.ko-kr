# <a name="argument-argumentname-cannot-be-converted-to-a-numeric-value"></a>인수 '\<argumentname>'을 숫자 값으로 변환할 수 없습니다.

문자열 등의 변수를 숫자 값으로 변환하려고 했습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. 의미 있는 변환을 찾습니다.
