# <a name="automation-object-does-not-have-a-default-value"></a>자동화 개체에 기본값이 없습니다.

Visual Basic는 지정 된 개체의 기본 멤버를 확인할 수 없습니다.  
  
### <a name="to-correct-the-error"></a>오류를 수정하려면  
  
- 개체의 설명서를 확인하고 메서드나 속성을 명시적으로 지정합니다.
