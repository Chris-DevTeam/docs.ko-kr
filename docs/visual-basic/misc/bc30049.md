# <a name="statementname-statement-requires-an-array"></a>' \<statementname> ' 문에 배열이 필요 합니다.

`Erase` 및 `ReDim` 문에 배열 데이터 형식이 필요합니다. 제공된 형식은 배열이 아닙니다.  
  
 **오류 ID:** BC30049  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 데이터 형식을 배열 형식으로 변경합니다.
