# <a name="specifier-is-not-valid-on-an-enum-declaration"></a>' \<specifier> '은 (는) 열거형 선언에서 사용할 수 없습니다.

`Enum` 선언에서 유효하지 않은 지정자를 사용했습니다.  
  
 **오류 ID:** BC30396  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. 지정자를 제거합니다.
