# <a name="shared-sub-new-cannot-be-declared-specifier"></a>공유 ' Sub n e w '는 ' '로 선언할 수 없습니다. \<specifier>

`Shared Sub New` 선언에서 지정자를 사용하려고 했는데, 이는 잘못된 것입니다.  
  
 **오류 ID:** BC30480  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 지정자를 제거합니다.
