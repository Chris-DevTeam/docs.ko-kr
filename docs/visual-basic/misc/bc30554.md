# <a name="namespacename-is-ambiguous"></a>' \<namespacename> '이 모호 합니다.

모호한 이름을 제공했으므로 다른 이름과 충돌합니다. Visual Basic 컴파일러에는 충돌 해결 규칙이 없습니다. 이름을 명확 하 게 구분 해야 합니다.  
  
 **오류 ID:** BC30554  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 이름을 정규화합니다.
