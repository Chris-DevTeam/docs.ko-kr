# <a name="attributes-cannot-be-applied-to-local-variables"></a>지역 변수에는 특성을 적용할 수 없습니다.

지역 변수에는 특성을 적용할 수 없습니다.  
  
 **오류 ID:** BC30660  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. 클래스 또는 모듈 수준과 같이 다음 상위 수준의 범위에서 이 변수를 선언 합니다.
