# <a name="addressof-expressions-are-not-valid-in-debug-windows"></a>디버그 창에서는 'AddressOf' 식을 사용할 수 없습니다.

`AddressOf` 문은 소스 코드에서만 사용할 수 있습니다.  
  
 **오류 ID:** BC30731  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. 디버깅 코드에서 `AddressOf` 문을 제거합니다.
