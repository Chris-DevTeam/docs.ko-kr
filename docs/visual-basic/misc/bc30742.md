# <a name="value-valuename1-cannot-be-converted-to-valuename2"></a>' \<valuename1> ' 값을 ' ' (으)로 변환할 수 없습니다. \<valuename2>

호환되지 않는 두 형식 간에 형식을 변환하려고 했습니다.  
  
 **오류 ID:** BC30742  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 캐스트 중인 형식을 확인합니다.
