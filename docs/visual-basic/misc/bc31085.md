# <a name="date-constant-is-not-valid"></a>날짜 상수가 잘못되었습니다.

날짜 상수가 잘못된 날짜로 선언되었습니다.  
  
 **오류 ID:** BC31085  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 상수에 사용된 날짜를 수정합니다.
