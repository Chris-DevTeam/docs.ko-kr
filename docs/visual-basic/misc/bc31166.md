# <a name="expected-closing--for-xml-tag"></a>XML 태그에 닫는 ' > '가 필요 합니다.

XML 리터럴의 XML 태그에 필요한 닫는 '>' 문자가 없습니다.  
  
 **오류 ID:** BC31166  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- XML 리터럴의 XML 태그 끝 부분에 닫는 '>' 문자를 추가합니다.
