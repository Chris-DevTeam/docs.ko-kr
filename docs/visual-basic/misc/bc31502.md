# <a name="resource-name-resourcename-cannot-be-used-more-than-once"></a>리소스 이름 '\<resourcename>'은 두 번 이상 사용할 수 없습니다.

동적 연결 라이브러리 또는 실행 파일의 지정된 리소스에 대한 이름이 두 번 이상 사용되었습니다.  
  
 **오류 ID:** BC31502  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 리소스 이름을 고유하게 지정합니다.
