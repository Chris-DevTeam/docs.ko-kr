# <a name="at-least-one-parameter-type-of-this-binary-operator-must-be-the-containing-type-typename"></a>이 이항 연산자의 매개 변수 형식 중 적어도 하나는 포함 하는 형식 ' ' 이어야 합니다. \<typename>

이항 연산자의 정의가 해당 연산자가 정의된 클래스 또는 구조체의 형식이 아닌 형식을 사용하여 두 매개 변수를 지정합니다.  
  
 클래스 또는 구조체에서 연산자를 정의할 때 매개 변수 중 하나 이상은 해당 클래스 또는 구조체의 형식이어야 합니다.  
  
 **오류 ID:** BC33021  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 매개 변수 하나 또는 두 형식 모두를 연산자가 정의된 클래스 또는 구조체의 형식으로 변경합니다.
