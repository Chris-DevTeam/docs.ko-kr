# <a name="before-and-after-arguments-cannot-be-combined"></a>'Before'와 'After' 인수는 함께 사용할 수 없습니다.

인수 목록에는 상호 배타적인 `Before` 및 `After` 인수가 모두 포함됩니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. `Before` 또는 `After` 인수를 제거합니다.
