# <a name="databits-must-be-greater-than-0"></a>DataBits는 0보다 커야 합니다.

`DataBits` 메서드에 지정된 `My.Computer.Ports.OpenSerialPort` 인수는 0보다 커야 합니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- `DataBits` 인수 값을 양수로 변경합니다.
