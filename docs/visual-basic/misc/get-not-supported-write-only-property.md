# <a name="get-not-supported-write-only-property"></a>Get은 지원되지 않습니다(쓰기 전용 속성).

쓰기 전용이기 때문에 속성을 읽을 수 없습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. 속성을 확인하고 설정할 수 있는 조건을 결정합니다.  
  
2. 속성에 대한 참조를 제거합니다.
