# <a name="on-visual-basic"></a>On(Visual Basic)

런타임 오류에 대 한 응답을 제공 하거나에서 컴파일러 옵션을 설정 합니다.  
  
 쿼리에서 조인 되는 컬렉션에 대 한 키를 식별 합니다.
