# <a name="search-text-not-found"></a>검색 텍스트를 찾을 수 없습니다.

지정한 텍스트를 찾을 수 없습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. 지정한 텍스트의 철자가 맞는지 확인합니다.
